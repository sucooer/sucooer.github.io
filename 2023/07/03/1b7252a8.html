🎨安装

  • 安装插件，在博客根目录打开终端运行
  • 在站点配置文件_config.yml添加配置信息
  • 执行hexo三连

🎨自建Api

  • 登录自己的Github账号并Fork仓库

    GitHub - zkeq/python_github_calendar_api: 用python获取github上的用户贡献信息，部署于vercel的api

  • 直接使用vercel部署并绑定自己的域名

  • 验证Api是否有效
