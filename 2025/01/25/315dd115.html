自建Meting API添加网易云音乐cookie

发表于2025-01-25
字数总计:4k阅读时长:23分钟阅读量: 泸州评论数:
技术分享Meting

自建Meting API添加网易云音乐cookie

若雨2025-01-25


前言

一直以来 MetingJS都只能解析免费音乐，不过我们可以通过自建 Meting API 并添加自己的会员账号 cookie 来解析 vip 歌曲。

Vercel部署

点击链接直接在 vercel 部署，它会在你的 github 账号创建一个同名仓库。因为涉及到设置 cookie，所以我们设置成私人仓库。

vercel部署Meting

等待部署完成后访问域名即可查看是否成功，vercel自带的域名被墙了，建议自己添加域名。

添加cookie

找到 github 仓库的src/providers/netease/util.js文件，删除原来的内容，粘贴下方代码，在提示位置粘贴网易云音乐的账号cookie，保存并提交，vercel会自动重新部署。

如果没有报错，这时候就能解析 vip 歌曲了，Meting API的使用方法具体可查看官方文档，为了方便调用，还可以将 API 嵌入 Metingjs。

API嵌入Meting.min.js

下载 Meting.min.js保存到设备上

下载源码

编辑此文件，Ctrk + F 查找 https://api.i-meto.com/meting/ 替换成自建 API 的自定义域名,Ctrl + S 保存。替换后就可以放到代码空间里使用链接引用此文件了，或放到本地使用相对链接引用也可以，这样以后使用 MetingJS 都可以解析 vip 歌曲了。
